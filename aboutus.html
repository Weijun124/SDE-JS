How to Use the Website

Welcome to our website. We are so glad that you are here. Restaurant Scanner is an innovative website using the Yelp API to help you find Restaurants in your area and search by location. It minimizes all the noise and advertisements that Yelp.com tends to have but all of the necessary information that you still want.

We feature a modern and simplistic view that any one can use. You can search by location, category, price level, and wheelchair accessibility.
